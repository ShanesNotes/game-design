# Prompt attic

Retired build-phase prompts (ADR 0006). The factory's terminal artifact is a spec
pack; building, playtesting, bakeoffs, content, art, polish, QA, and release happen
in the exported co-dev folder, not here. These prompts are kept as doctrine a
co-dev session may borrow — they are NOT part of the live contract
(`scripts/lib/factory-contract.mjs` PROMPTS) and no skill routes to them.
